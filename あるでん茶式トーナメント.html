インターネットバックギャモンクラブ（INBC）公式ホームページ

あるでん茶式トーナメント

概要

ラウンド1からラウンド3までを予選ラウンドのような扱いにしたトーナメントです。ラウンド3終了時に2r人が勝ち残り、そこからは普通の1敗失格式トーナメントとなります。Single eliminationとDouble eliminationの中間で、Semi-double eliminationと呼ぶことにしようと目論んでいます。

以下のような特徴を持ちます。
  • 全ての参加者は、最初の試合で敗れても敗退とはなりません。すなわち全ての参加者に2試合を保証します。
  • 抽選による枠の有利不利の差が、通常の1敗失格式トーナメントに比べて小さくなることが多いです（参加人数に依存します）。
  • 参加人数に応じたテンプレートを用意することにより、最初に一度抽選を行うだけであとの対戦は自動的に決まります。

用途はバックギャモンに限るものではありませんが、番狂わせが起こりやすいバックギャモンと相性はいいと思います。特に（基本的に参加者の立場がフラットな）INBCの1日完結大会では上述の特徴がいい方向に作用すると思います。

あるでん茶が独自に開発したものだと信じていますが、もしかしたら再発明かもしれません。
  • トーナメントの平等度：トーナメントの平等度を評価して、あるでん茶式と一敗失格式を比較しています。

テンプレート

Googleスプレッドシートとして公開しています。現在、5人から42人まで対応しています。

理論的実装？

枠数の割り当て

全ての試合の勝率が1/2である場合に、i/8 の確率でラウンド4に進める（=i/2r+3の確率で優勝できる）枠のことを「i枠」と呼びます。

  • 参加人数Nが確定すると、Round4進出者数2rを、次式で決定します。 r = min{r | 3N+1 < 2r+3}
    • このとき、 3N+1 < 2r+3 <= 4N, 4N < 2r+3 < 5N, 5N < 2r+3 < 6N, 2r+3 = 6N+2 のいずれかが成立します。
  • 以下のアルゴリズムにより、各参加者に割り当てるべき枠を用意します。
    1. 3N+1 < 2r+3 <= 4N かつNが偶数の場合、N人に対し1人3枠または4枠を、合計2r+3枠になるように割り当てます。いわゆる鶴亀算です。
    2. 3N+1 < 2r+3 < 4N かつNが奇数の場合、1人に6枠を、残りの(N-1)人に対し1人3枠と4枠を、合計2r+3-6枠となるように割り当てます
    3. 4N < 2r+3 < 5N かつNが偶数の場合、N人に対し1人4枠または5枠を、合計2r+3枠になるように割り当てます。
    4. 4N < 2r+3 < 5N かつNが奇数の場合、1人に6枠を、残りの(N-1)人に対し1人4枠と5枠を、合計2r+3-6枠となるように割り当てます。
    5. 5N < 2r+3 < 6N の場合、N人に対し1人5枠または6枠を、合計2r+3枠になるように割り当てます。
    6. 2r+3 = 6N+2 の場合、2人に対して7枠を、残りの(N-2)人に対して6枠を割り当てます。
  • 参加者の枠への割り当ては基本的には抽選で行いますが、何らかの実績を持つ参加者に対して優遇措置を与えることも可能です。
    • 飛び抜けて有利な1枠が生じることがあるので、通常その優遇措置は1人に対して与えられます。

上記bの状況でのみ、参加者間に最大2倍の格差が生じます。bとd以外では、参加者間の最大格差は1.5倍未満になります。
（仕様上、4枠を持つ参加者を偶数にする必要があります。bのケースで一見不自然な6枠が1つ生じているのは、参加者数のパリティの調整を6枠で行うしかないという事情によります。）

トーナメントの組み立て

あとは、適当に組み合わせます。
  1. 6枠の参加者を除いて、ラウンド1で同じ枠数の参加者同士が対戦します。
    • 勝つと対戦相手から1枠を奪います。
    • ただし4枠同士の対戦では2枠を奪います。
  2. ラウンド2では、（ラウンド1の増減の結果）2枠と6枠を持つ参加者が対戦します。基本的には同じ枠数同士の対戦となりますが、6枠vs2枠の対戦が生じることもあります。
    • 勝つと対戦相手から2枠を奪います。8枠になるとラウンド4に進出します。0枠になると敗退です。
  3. ラウンド3では、（ここまでの増減の結果）4枠をもつ参加者が対戦します。
    • 勝つとラウンド4に進出し、敗れると敗退となります。

なお、2度目の同じ対戦が生じることがあります。それがなるべく後のラウンドになる確率を最大化するように実装したつもりですが、特に理論立ててはいないので、そうなっていないかもしれません。
